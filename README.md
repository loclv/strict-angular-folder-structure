## Run mock API from swagger file üß™

Use [this tool](https://meta.stoplight.io/docs/prism/README.md).

```sh
yarn api
```

or if you want to using in global context:

```sh
npm install -g @stoplight/prism-cli
prism mock ./swagger.yaml
```
